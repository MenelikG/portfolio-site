Mentr App

The Mentr App is a web application that helps match potential mentors with mentee's based on common interests and technical skills. I developed this project with a team of four during my six-month experience in the Capital One Developer Academy.

Technologies Used: MongoDB, Angular, Express, Node.js, Bcyprt.js, Sockets.io
Features: Login Authentication, Instanst Messaging, Gestures
