Mentr App

The Mentr App is a web application that helps match potential mentors with mentee's based on common interests and technical skills. I developed this project, along with three others, during my six-month experience in the Capital One Developer Academy. We made the UI of the application resemble Tinder, Bumble, and similar dating apps because we believe the "swipe left/right" design is most user-friendly and has become familiar to a lot of young professionals.

Technologies Used
  • MongoDB
  • Angular
  • Express
  • Node.js
  • Sockets.io
  • Bcrypt.js
Features
  • Instant messaging
  • User Authentication
  • Gestures
